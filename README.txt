Чтобы создать новую службу windows, необходимо выполнить команду от имени администратора: sc create MyService binPath= "путь_к_вашему_приложению",
где MyService - это имя будущей службы, а путь_к_вашему_приложению - это путь к исполняемому файлу вашего Go-приложения;
Затем следом введите команду: sc start MyService, чтобы запустить службу на windows;
Удалить службу, ввести команду: sc delete MyService.

После того, как служба будет запущено введите в командной строке такую строку - curl -X GET localhost:8080/calculator
Данная команда выведет небольшое сообщение, это значит, что служба работает и готова выполнять все необходимые действия пользователя
в зависимости оттого, какой код пользователь впишет для выполнения той или иной команды.
Если будет введена другая команда, выведется сообщение об ошибке.
